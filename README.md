# Better Voice Chat BepInEx Plugin
A mod for Casualties: Unknown that improves multiplayer voice chat.

## Features
- Toggleable background noise suppression
- Toggleable automatic gain control (makes quiet mics/whispers louder and vice versa)
- Configurable automatic gain control target (temporary solution for additional volume control)

## Notes
- Not natively compatible with Linux (yet). Should run fine under a compatibility layer (hopefully).
